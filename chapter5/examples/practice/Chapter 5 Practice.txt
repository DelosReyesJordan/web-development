1. Create a heading and change the font family

2. Create a paragraph of text and change the font weight for it

3. Move the paragraph to the center of the webpage


ANSWERS:

HTML:
<!DOCTYPE html>
<html>
    <head>
    	<link rel="stylesheet" type="text/css" href="FILE_PATH_NAME.css">
    </head>
    <body>
    	<div class = "text">
        	<h1 id = "title">Hello World</h1>
        	<p id = "para"> This is a paragraph of text. Try changing or adding css to the css code and experiment with how this paragraph changes. </p>
        </div>
    </body>
</html>

CSS: 

#title{
	 font-family: 'Raleway', sans-serif;

}
#para{
	font-weight: 200;
	left: 0;
    line-height: 200px;
    margin-top: -100px;
    position: absolute;
    text-align: center;
    top: 50%;
    width: 100%;
	/* jsfiddle: http://jsfiddle.net/wprw3/ */
}
